## ðŸ”– Como executar

#### Clonando o projeto
```sh
git clone https://github.com/wesleyfeitosa/gerenciador-de-estabelecimentos.git
cd gerenciador-de-estabelecimentos
yarn install
```
#### Iniciando servidor
```sh
cd server
yarn start:docker
yarn start:dev
```
#### Starting Frontend
```sh
cd web
yarn start
```
#### Starting mobile
```sh
cd mobile
yarn start
```
