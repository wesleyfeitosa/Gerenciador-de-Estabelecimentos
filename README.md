## ðŸ”– Como executar

#### Clonando o projeto
```sh
git clone https://github.com/wesleyfeitosa/gerenciador-de-estabelecimentos.git
cd gerenciador-de-estabelecimentos
```
#### Iniciando servidor
```sh
cd server
yarn install
yarn start:docker
yarn start:dev
```
#### Iniciando frontend web
```sh
cd web
yarn install
yarn start
```
#### Iniciando app mobile
```sh
cd mobile
yarn install
yarn start
```
